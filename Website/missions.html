Before you pack that amazing bread you stole, let's first go over what missions are all about.

So far, only short missions are implemented.

A short mission takes 5 minutes to complete and you will be notified of the result after that time expires.

To start a mission, use the mission command and pass a length argument into it. Currently, only 'short' is supported.

@Justine#1053 mission start

GG, did I win?

Keep in mind adventuring is a demanding task and it might very well kill you. Make sure you evaluate the risk before embarking on an adventure.

Medic!

If you need healing, you can buy 50 HP (health points) for 50 gold with the heal command.
