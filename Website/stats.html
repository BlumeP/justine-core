RPG Stats

'Stats' is referring to the collection of your character's attributes.

These attributes are used by many RPG features, such as missions or gold digging.

If you've already given your consent to data collection, you can view your character's stats with the following command:

@Justine#1053 stats

Am I better than my friends?

To see someone else's stats, you can overload the stats command with a user mention.

@Justine#1053 stats @YouFriend#001

Upgrading stats

To upgrade a stat, we use the upgrade command which takes in the short name of a stat, which can be found in the stat listing in square brackets.

Upgrading a stat costs you gold depending on the level. You can use stats command to see the needed amount.

Variations

@Justine#1053 upgrade STR

@Justine#1053 upgrade SPD

@Justine#1053 upgrade INT

@Justine#1053 upgrade END

@Justine#1053 upgrade LCK
